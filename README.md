# Hackathon Helper Tool - Frontend

A lightweight Kanban board application built with React, TypeScript, and Tailwind CSS. This is the frontend part of a MERN stack application designed to help hackathon teams organize tasks and share progress.

## Features

- **Drag & Drop Kanban Board**: Intuitive task management with visual columns
- **Task Management**: Create, assign, and prioritize tasks
- **Team Collaboration**: Track team members and task assignments  
- **Priority System**: Visual priority indicators (High, Medium, Low)
- **Dark Mode**: Beautiful dark theme with smooth transitions ðŸŒ™
- **Responsive Design**: Works on desktop and mobile devices
- **Real-time Updates**: Seamless task movement between columns

## Tech Stack

- **React 19** with TypeScript
- **Vite** for fast development and building
- **Tailwind CSS** for styling
- **@dnd-kit** for drag and drop functionality
- **Lucide React** for icons

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

    },
